# reversi

## Запуск

"reversi -h" - помощь
"reversi -p -f [-d=depth]" - ходить первым, depth = 6 по умолчанию
"reversi -p -s [-d=depth]" - ходить вторым

## Игра

Сделать ход - написать клетку [a..h][1..8]
Если хода нет - "/Skip 0"
Дополнительные опции - "/Help"

## Алгоритмы

Использовались альфа-бета отсечения.
